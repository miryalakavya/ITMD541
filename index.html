My Web Projects

Blood Donation

My first website was a project for blood donation. Through this website, acceptors could access blood quickly. It offered subscription options, and users could register through the dedicated registration page.

Holiday Calendar

This project was built using React.js. The website checks if the current day is a holiday and displays an appropriate message accordingly.

Chat Hub

This project is reminiscent of a social media platform like Facebook. Built with HTML, CSS, and JavaScript, it enables users to sign in/out, create and delete posts, like and comment on posts. It also features a chat functionality and a Google form through which users can submit data, which is then sent to the admin's drive.

Phonebook Web App

Developed using Node.js, this web app allows users to add contact details with multiple contact numbers and emails, edit contacts, and offers functionalities like pagination and searching by name. It's deployed on Heroku, with a database instance from MLAB.
